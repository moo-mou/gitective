Gitective

Record your daily activities and grade yourself.

What is it?

This is a simple personal task tracker that allows you to create a set of tasks to track and gives you a grade based on what you record each day.

See an example

Note - If it doesn't work the first time, try refreshing as there is connection limits by Firebase

How do I use it?

Getting started is easy - simply choose a username and click on the CREATE button below.

Important - Right now, Gitective creates a unique URL after you create an account and this URL is the only way for you log back to the same user. Anyone with the URL can see your data so only share it people you trust. Bookmark it!
